# Fisq <img src="https://franekk3.github.io/Fisq/icons/icon.png" height="50px">
Fisq! - free, open-source, web based flashcards!
### Futures:
- Free to use
- No trackers
- Open Source
- Cross Platform (Web Based)
- Offline Android App
### Building from source:
##### Web:
- just clone repo and start simple http server in main directory

*for example:* `python3 -m http.server`
##### Android:
*NOTE: Android App is just WebView container that wrapps site files from local storage (works compleatly offline)*
- Clone repo
- Open `android-app` folder in **Android Stuido**.
- Build using **Android Studio**
