# **Fisq!** is a completely **free** Android app made for learning using FlashCards.

In‑app features:
- no paywalls
- Material 3–like UI - user friendly
- no diagnostic data sent anywhere
- built‑in creator & editor for your own flashcard sets
- less than 10 MB .apk file! (compatible with ARMv7 too)
- Fisq! futures dark mode too

> This is one of the first public builds of the app (first public one), so it may contain some bugs and unfinished features. Stay tuned!
