FAQ

Is Fisq! Open Source? Yes, Fisq! is Open Source and it's whole Sorce Code is avible on it's Github repo
Is Fisq! free? Yes, Fisq! is compleatly free to use without any ads!
Does Fisq! store my data on external servers? No, all your data is stored locally in your browser's data.
How can I use Fisq? You can use Fisq! in your browser here but you can also download offline app for Android here .

Building from source:

1. Web

a. clone repo from here
b. open downloaded projects folder
c. run http server in main directory

for example: python3 -m http.server
commend above requies python installed

2. Android

Option A

a. clone repo from here
b. go to android-app folder inside downlaoded project, using Android Studio
c. build using Android Studio

requies Android Studio installed

Option B

a. clone repo from here
b. go to android-app folder inside downlaoded project
c. build using ./gradlew build #linux/mac or ./gradlew.bat build #windows in terminal/cmd
